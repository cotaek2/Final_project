## 👨‍🏫 프로젝트 소개
### DFLIX
"영화 예매를 위한 스프링3.0 MVC 기반 사이트입니다.
<br>
불필요한 시스템을 줄여 강의 및 학사관리에 효율을 극대화"

## ⏲️ 개발 기간 
- 2024.3.18. ~ 2024.4.16.<br>
  
## 🧑‍🤝‍🧑 개발자 소개 
- **이준택** : 팀장, 프로그램 관리, 기획, 검토
- **전용휘** : 팀원
- **정다희** : 팀원

## 💻 개발환경
- **Tomcat 9.0**
- **MySQL 8.0.35**
- **JDK 13.0.2**
- **GitHub**
- **AWS**

## 🌐 개발 언어
- **HTML**
- **CSS3**
- **JavaScript**
- **jQuery**
- **JAVA(JSP, JDBC)**
- **Spring Framework 4.3.4**

## ⚙️ 개발 도구
- **STS 3(Spring Toll Suite)**
- **StarUML**
- **ERMaster**
- **FileZila**

## 🚀 가상화 플랫폼
- **Docker**
